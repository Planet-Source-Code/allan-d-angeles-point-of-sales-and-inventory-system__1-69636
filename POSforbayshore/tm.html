-== Transaction Module ==-

Sub module:

  • Sales
  • Delivery
  • Orders
  • Return
      - Return from customer
      - Return to supplier

 

Sales - Allow the user to operate sales transaction and this form can produce an official receipt for the customer

Delivery  - Allow the user to operate delivery transaction.

Orders - Allow the user to order op products to their supplier.

Return from customer - Perform all product replacement and display returned product from customer.

Return to Supplier - Performa all product replacement and  product to be returned to their supplier.
